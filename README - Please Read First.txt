SETUP GUIDE (Sirf 3 Steps!)
============================================================

STEP 1: SETUP.bat double-click karo
   → Python install hoga
   → Sab dependencies set honge
   → secrets.txt khul jayegi

STEP 2: secrets.txt mein apni details bharo
   → DELTA_API_KEY    = (Delta Exchange se lena)
   → DELTA_API_SECRET = (Delta Exchange se lena)
   → TELEGRAM_TOKEN   = (BotFather se lena)
   → TELEGRAM_CHAT_ID = (Apna Telegram ID)

STEP 3: RUN_ALL.bat double-click karo
   → Bot + Dashboard ek saath start
   → Dashboard: http://localhost:8600

============================================================
DELTA EXCHANGE API KEY KAISE BANAO:
============================================================
1. www.delta.exchange pe login karo
2. Account > API Management
3. "Create API Key" dabao
4. Name: BharatFutures
5. Permissions: Read + Trade
6. IP Address: Apna IP daalo (whatismyip.com se dekho)
7. Key + Secret copy karo → secrets.txt mein daalo

============================================================
TELEGRAM BOT KAISE BANAO:
============================================================
1. Telegram mein @BotFather search karo
2. /newbot command do
3. Bot ka naam do → Token milega → secrets.txt mein daalo
4. Apna Chat ID: @userinfobot se milega
